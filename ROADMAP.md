# Project Roadmap

Welcome to the **ELONODE** roadmap! Here is a high-level overview of what we are currently working on and what we plan to build next.

## Phase 1: Core Arena (Current)

- [x] Basic Next.js + Tailwind CSS UI setup
- [x] Go backend architecture with GORM
- [x] 3v3 ICPC mode match creation form
- [x] Standard ICPC penalty and leaderboard computation logic
- [ ] Live Match Arena UI (Editors, Problem viewer)

## Phase 2: Real-time & Multiplayer

- [ ] WebSockets integration for live leaderboard updates
- [ ] Real-time typing/cursor sync for team members
- [ ] 1v1 Duel Mode system

## Phase 3: Global Ecosystem

- [ ] Elo Rating system for users
- [ ] Global Leaderboards (Top ranks: Grandmaster, Master, etc.)
- [ ] Match History and Player Profiles
- [ ] Automated code execution engine (Judges)
